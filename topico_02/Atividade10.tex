Isto NÃO é um comentário, mas é ignorado pelo LaTeX porque ocorre após \end{document}
(qualquer coisa colocada após \end{document} é ignorada). Mas não faça deste um espaço
para comentários. Ao invés disso, use % explicitamente, como acima.
