	% A construção do conteúdo de uma tabela longa é exatamente a mesma de qualquer outra. A diferença está no início, onde algumas instruções extras devem ser informadas. A saber, é preciso informar quais serão os cabeçalhos e rodapés necessários. O esquema abaixo ilustra quais informações são necessárias:

	%  Página 1             Página 2                 Página 3
	% 
	%  ----------------     --------------------     ----------------------
	% | Cabeçalho p. 1 |   | Cabeçalho p. 2 e 3 |   | Cabeçalho p. 2 e 3   |
	% | ...            |   | ...                |   | ...                  |
	% | ...            |   | ...                |   | ...                  |
	% | ...            |   | ...                |   | ...                  |
	% | ...            |   | ...                |   | ...                  |
	% | Rodapé p. 1 e 2|   | Rodapé p. 1 e 2    |   | Rodapé p. 3 (ÚLTIMA) |
	%  ----------------     --------------------     ----------------------
	%
	% Imagine que tenhamos uma tabela que ocupe três páginas. Neste caso precisamos informar ao pacote os seguintes cabeçalhos e rodapés:
	%
	% Cabeçalho da página 1
	% Cabeçalho da página 2 em diante
	% Rodapé da página 1 até a penúltima
	% Rodapé da última página
