% A função da sequência de comandos \noindent\hrulefill\par é apenas produzir
% uma linha contínua de uma margem a outra do texto para evidenciar o avanço
% do parágrafo seguinte para além da margem direita. Ela funciona assim: o
% comando \noindent (de "no indentation") suprime a endentação do parágrafo,
% enquanto o comando \hrulefill preenche ("fill") a linha do texto com uma
% linha ("rule") horizontal ("h"). O comando \par você já conhece: ele promove
% a mudança para o próximo parágrafo.
